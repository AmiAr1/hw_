 CREATE, INSERT, SELECT, UPDATE, DROP



1. Оператор SQL CREATE служит для создания объектов базы данных.
Разные СУБД работают с разными объектами,
но наиболее общими для большинства СУБД будут команды создания таблицы (оператор SQL CREATE TABLE)
и создания базы данных.


2. INSERT - оператор языка SQL, который позволяет добавить строки в таблицу, заполняя их значениями.
Значения можно вставлять перечислением с помощью слова values и перечислив их в круглых скобках через запятую


3. SELECT - в языке SQL, возвращающий набор данных (выборку) из базы данных.
Оператор возвращает ноль или более строк.
Список возвращаемых столбцов задается в части оператора, называемой предложением SELECT.


4. UPDATE - оператор языка SQL, позволяющий обновить значения в заданных столбцах таблицы.


5. DROP TABLE удаляет таблицы из базы данных. Удалить таблицу может только её владелец,
владелец схемы или суперпользователь
